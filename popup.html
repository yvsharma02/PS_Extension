BITS PS2 Helper

DISCLAIMER: Although I've tried by best to extensively test and solve any issue I encountered, 100% saftey cannot be garunteed. Please always double that check the info shown by the extension. Also double check any preference-list you import or export has been done so correctely.
I shall not be liable for any failure of the extension, or any mishap happening due to this extension.

To start using the extension, go to the following page, and refresh (press F5):
https://psms-web.azureedge.net/stationpreference/selectpreference
Upon reloading, a pannel containing details of the currently selected station will be inserted at the top of the page.
More controls for easier list sorting, and importing/exporting list will also be present in the pannel.
